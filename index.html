Fully Autonomous Kissing Robot

- Used Python for the kissing app (which I made via VS Code on my laptop)

- Then turned it into a Linux executable so it could run on the Raspberry Pi board (the board is attached behind the screen.

- I bought the screen online, and made sure it was compatible w/ my specific Raspberry Pi. Also, the package came w/ instructions for how to connect it)

- Set up a separate Arduino project using an Arduino Uno, a servo motor, a sensor, and a board attached on top of the Arduino (an L293D Drive Module Motor)

- Soldered header pins to the L293D to get some wires connected from the sensor and the servo motor

- Got some motors and wheels from an Arduino kit I found online, to set up the robot's movement, and attached those wires to the L293D as well

- After all was said and code was pushed to the Arduino Uno, I connected the Raspberry Pi on top of the actual moving Arduino robot via hot glue.

- I also added some supporting axles at the bottom to make sure the robot could support its own movement

- Made it so the sensor moves from left to center to right to sense any objects before backing up out of the objects' way, and slightly turning right

- Sound is quiet since I used some quiet headphones, but the Linux executable asks the user for a kiss.

- When the user presses the button presented, a “thank you” message appears, and the app says “thank you so much.”

- Lastly, I used a power bank to power both devices via their respective power cables
